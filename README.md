2. 运行 `yarn start:dev` 进行测试开发

## 文件结构 ##
* /_next/static/style.css  `配置next-less、next-css所需要的对应路径文件`

* /pages `页面`

* .gitignore  `git配置文件`
* package.json  `包配置信息`
* readme.md  `此文件`
* tsconfig.json  `typescript编译配置`
* yarn.lock  `yarn生成的包依赖信息`
